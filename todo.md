# 功能
P1：支持中英文切换，默认根据浏览器语言进行切换。
P1：设置页面优化
P1：目录点击改成toggle行为，点一次展开，再点一次收起
P1：页面要根据图片的实际尺寸适配每个图片卡片的区域，上下左右都不能留太大的空白。如果是竖图，宽度至少350px，如果是横图，高度至少350px

P1：非幻灯片状态下时，方向键用于操作目录浏览：上下用于切换目录。右方向用于展开目录，左方向用于收起目录。同时要在配置中支持配置快捷键。同时，空格键进入工作状态，空格键退出工作状态，也要支持配置快捷键。

# bug

P1：对同一张图片不能进行多次操作，否则第二个操作会报错
P1：设置中，捕获按键状态没有正确展示。点击捕获后，状态要展示成请按按键的明显效果。
